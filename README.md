# type conversion (conversão de tipo) type coercion (correção de tipo)

```js
   var x = Number("0") //Type Coercion, nessa linha é feita uma correção de tipo explicita

   var y = "2" +3 // Tipo Coercion, nessa linha é feita uma correção ixplicito do valor 3 que é númerico(number) para texto(string)

//Onde uma expressões condicional e repetição, faz necessário um valor boolean ( verdadeiro ou falso)
//Ex.
if(true) {
}
//Caso não for informada uma expressão ou um valor boolean diretamente, ele fará um Type Coercion (Correção do tipo)
//Ex.
if(0) ( //Nessa linha ela ira fazer um correção do 0 para um boolean (verdadeiiro ou falso), nesse caso 0 será falso
]
```
